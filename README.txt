# Cálculo de Factor de Diseño

## Descripción
Este programa calcula el factor de diseño y el diámetro requerido para un eje sometido a esfuerzos mecánicos utilizando factores de concentración de esfuerzo.

El algoritmo utiliza tablas de factores Kt y Kts para determinar el comportamiento del material bajo cargas aplicadas.

## Funcionamiento
El programa solicita o utiliza datos de entrada como:

- Esfuerzo aplicado
- Propiedades del material
- Factores de concentración de esfuerzo
- Condiciones de carga

Con estos datos se calcula:

- Factor de diseño
- Diámetro necesario del eje
